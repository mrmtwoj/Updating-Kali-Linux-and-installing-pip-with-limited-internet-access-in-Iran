# Updating-Kali-Linux-and-installing-pip-with-limited-internet-access-in-Iran
این مخزن یک راهنمای جامع برای نصب و به‌روزرسانی Kali Linux و ابزارهای پایتون مثل pip در شرایط اینترنت محدود ایران است. در این پروژه روش‌های رفع خطاهای apt و pip، انتخاب مخازن مناسب، تنظیم DNS، افزایش سرعت دانلود بسته‌ها و راهکارهای دور زدن محدودیت‌های شبکه ارائه شده است.
